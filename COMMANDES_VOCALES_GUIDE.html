📖 Introduction

🎯 Principe de fonctionnement

Toutes les commandes vocales commencent par le mot-clé "OK" suivi de l'instruction. Le système utilise une validation stricte pour garantir que tous les champs obligatoires sont remplis.

1
Mot-clé de réveil
Dire "OK" pour activer l'écoute
2
Type d'action
Point gagnant, Faute directe, Faute provoquée, etc.
3
Joueur(s)
Nom du/des joueur(s) concerné(s)
4
Type de coup (si applicable)
Service, Smash, Volée, Fond de court, etc.
✅ Avantages
  • Annotation sans interrompre la vidéo
  • Validation stricte : aucun point incomplet
  • Feedback visuel immédiat (bandeau rouge si erreur)
  • Plus de 50 commandes disponibles
  • Interface intuitive et naturelle

⚔️ Fautes Provoquées

⚠️ Validation Requise
Champs obligatoires :
  • ✅ Type d'action : "faute provoquée"
  • ✅ Joueur attaquant (qui provoque)
  • ✅ Joueur défenseur/fautif (qui subit)
Faute Provoquée
OK faute provoquée [Attaquant] [Défenseur]
L'ordre est important : d'abord l'attaquant (qui provoque), puis le défenseur (qui fait la faute)
💡 Exemples :
• OK faute provoquée Arnaud Pierre
• OK faute provoquée Thomas Lucas
• OK faute provoquée Pierre Arnaud

✅ Validation Stricte

✅ Commandes VALIDES (acceptées)
✓ OK point gagnant Arnaud service
✓ OK point gagnant Pierre balle haute smash
✓ OK point gagnant Thomas fond de court coup droit
✓ OK faute directe Lucas
✓ OK faute provoquée Arnaud Pierre
❌ Commandes INVALIDES (rejetées)
Ces commandes seront bloquées et afficheront un bandeau rouge :
✗ OK point gagnant Arnaud
→ Manque : TYPE DE COUP
✗ OK point gagnant balle haute
→ Manque : JOUEUR + SOUS-TYPE
✗ OK point gagnant Arnaud balle haute
→ Manque : SOUS-TYPE (smash/bandeja/víbora)
✗ OK faute provoquée Arnaud
→ Manque : DÉFENSEUR
🔄 Que faire en cas d'erreur ?
1
Bandeau rouge apparaît
Indique les champs manquants
2
Vidéo en pause automatique
Pour vous laisser le temps de corriger
3
Dire "OK supprimer"
Pour annuler la tentative
4
Répéter la commande complète
Avec tous les champs requis
