# 🎾 PFPADEL Video Stats - Guide d'utilisation

## 🚀 Nouvelles fonctionnalités !

### 1. 🎚️ Sélecteur de vitesse
- **Vitesses disponibles** : 0.25x, 0.5x, 0.75x, 1x, 1.5x, 2x
- **Localisation** : En bas à droite des contrôles vidéo
- **Utilisation** : Sélectionnez la vitesse dans le menu déroulant
- **Cas d'usage** :
  - 0.25x - 0.5x : Analyse détaillée des mouvements
  - 1x : Vitesse normale
  - 1.5x - 2x : Revue rapide du match

### 2. 📊 Barre de progression ultra-précise
- **Localisation** : Juste au-dessus des informations vidéo
- **Fonctionnalités** :
  - Cliquez n'importe où pour sauter à ce moment
  - Glissez pour naviguer en continu
  - Indicateur visuel de position (poignée blanche)
- **Couleurs** :
  - Fond gris : Vidéo non lue
  - Bleu : Portion lue
  - Poignée blanche : Position actuelle

### 3. 🖥️ Mode plein écran
- **Activation** :
  - Bouton "⛶" en bas à droite
  - Raccourci clavier : `F` ou `F11`
- **Fonctionnalités** :
  - Vidéo en plein écran maximisée
  - Contrôles overlay en bas (apparaissent au mouvement de souris)
  - Barre de progression cliquable
  - Tous les contrôles de lecture disponibles
  - Timer visible
- **Sortie** :
  - Bouton "✕ Quitter" rouge
  - Raccourci : `Échap`, `F`, ou `F11`
- **Auto-masquage** : Les contrôles se cachent après 3 secondes d'inactivité

## ⌨️ Raccourcis clavier

| Touche | Action |
|--------|--------|
| `Espace` | Lecture / Pause |
| `→` | Frame suivante |
| `←` | Frame précédente |
| `F` ou `F11` | Plein écran |
| `Échap` | Quitter plein écran |

## 🎮 Workflow recommandé

1. **Charger la vidéo** (`📁 Charger Vidéo`)
2. **Configurer les joueurs** (`👥 Joueurs`)
3. **Navigation rapide** :
   - Utilisez la barre de progression pour sauter aux moments clés
   - Ajustez la vitesse selon vos besoins
4. **Analyse précise** :
   - Réduisez la vitesse (0.25x - 0.5x)
   - Utilisez frame par frame (`←` / `→`)
5. **Annotation** :
   - Cliquez sur le type d'action
   - La capture est automatique
6. **Mode plein écran** :
   - Activez `F` pour une meilleure visibilité
   - Annotez directement en plein écran
7. **Export** :
   - JSON pour les données brutes
   - HTML pour le rapport visuel

## 💡 Astuces

- **Navigation rapide** : Cliquez sur la barre de progression
- **Analyse ralentie** : Vitesse 0.25x + frame par frame
- **Revue match** : Vitesse 2x pour voir rapidement
- **Plein écran** : Idéal pour projeter sur grand écran
- **Contrôles masqués** : Bougez la souris pour les réafficher
